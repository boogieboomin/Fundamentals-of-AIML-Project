# Fundamentals-of-AIML-Project
My Python tool predicts exam scores based on study hours. I used Pandas to organize data, then a for-loop to find the average score increase per hour. A function, get_prediction, calculates the result using a base of 50. Finally, input() and float() allow users to get real-time results. It’s a simple, logical way to see how effort impacts grades!

## Project Overview

This project predicts a student's exam score based on the number of hours they study.  
It uses a simple data-driven approach to show how study time impacts performance.

The goal is to help students:
-  Plan their study time effectively  
-  Set realistic academic goals  
-  Understand effort vs performance
